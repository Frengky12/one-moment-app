# One Moment

Aplikasi kamera acara (disposable-camera experience) untuk pernikahan/event, didistribusikan lewat vendor (Wedding Organizer/fotografer) dengan branding mereka sendiri.

## Dokumentasi

- [PRD](docs/PRD.md) — requirement produk, desain, dan skema database
- [Stitch AI prompts](docs/stitch-prompts.md) — prompt untuk generate desain UI

## Stack

- Next.js + TypeScript
- Supabase (DB, Auth, Storage, Realtime)
- Midtrans (payment/subscription)
- Vercel (hosting)

## Status

Tahap perencanaan — belum ada kode aplikasi, masih di fase PRD & desain.
